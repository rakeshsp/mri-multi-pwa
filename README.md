# MRI Multi-Study PWA

Vite + React + Tailwind + PWA app for structured MRI reporting:
- Ankle (full 17-section high detail)
- Spine (high-detail structured template)
- Knee (high-detail structured template)
- Shoulder (high-detail structured template)

## Local usage

```bash
npm install
npm run dev
```

Then open the shown URL (default http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## GitHub / Vercel

1. Initialise a repo and push this project.
2. Import the repo in Vercel as a Vite project.
3. Each git push to main triggers auto-redeploy.
